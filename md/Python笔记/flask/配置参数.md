# 配置参数

#### 初始化参数

- import_name
- static_url_path
- static_folder，默认：'static'
- template_folder， 默认：'templates'

#### 配置参数

- app.config.from_pyfile('config.cfg')
- app.config.from_object()

#### 在视图用读取配置参数

- app.config.get()
- current_app.config.get()

#### app.run 的参数

- app.run(host='0.0.0.0')
